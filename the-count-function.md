### Query a count of the number of cities in CITY having a Population larger than 100,000.

Input Format

The CITY table is described as follows:

  CITY

| Field  | Type |
| ------------- | ------------- |
| id  | number  |
| name  | varchar(17) |
|countrycode | varchar2(3) |
|district | varchar2(20) |
|population  | number |
